## 简介

面向资源受限 ARM Cortex-M 平台的智能手表原型工程。硬件上集成 OLED 显示、RTC、陀螺仪/磁力计、旋转编码器和蜂鸣器；软件上采用分层架构（HAL → 驱动 → 库 → 服务 → 应用 → UI），目标实现计时/闹钟、Dino 小游戏、计步器和指南针等趣味应用。

完整的教学项目，涵盖从硬件选型到固件实现的全流程。

## 核心特性

- 分层软件架构（HAL / 驱动 / 库 / 服务 / 应用 / UI）
- OLED 显示驱动
- RTC 时钟 / 闹钟
- Dino 小游戏
- 计步器（陀螺仪）
- 指南针（磁力计）
- 旋转编码器交互

## 前置知识

- ARM Cortex-M 开发基础
- C 语言编程
- 基本电子学知识

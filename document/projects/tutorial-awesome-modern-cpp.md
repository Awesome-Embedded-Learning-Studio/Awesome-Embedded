## 简介

10 卷体系现代 C++ 系统教程，从 C 语言基础一直覆盖到 C++23 最新特性。不同于碎片化的单篇教程，它将语言特性、标准库、工程实践和领域应用整合为一条完整学习路径，并针对嵌入式开发者特别优化了 STM32F1 实战环节。

配套 VitePress 文档站支持中英双语，所有示例代码均可通过 CMake 一键编译。

## 核心特性

- 10 卷渐进式内容：C 基础 → 现代 C++ → STL → 进阶 → 并发 → 性能 → 工程化 → 领域应用 → 开源研读 → 课程笔记
- 中英双语文档（英文翻译 98%+）
- CMake 编译验证的示例代码
- 嵌入式方向特别章节（STM32F1 实战）
- C++ 特性速查卡

## 前置知识

- 基本编程概念（变量、循环、函数）
- 命令行基本操作

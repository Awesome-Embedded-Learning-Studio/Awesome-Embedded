## 简介

面向 Rockchip 平台的开发者工作空间。提供脚本库、驱动参考实现、结构化补丁管理和 rootfs 部署方案，覆盖 RK3588/RK3568/RK3528/RK3506 等 SoC。

帮助开发者解决厂商 SDK 定制、内核补丁管理、驱动开发和多发行版 rootfs 部署的常见问题。

## 核心特性

- 支持 RK3588 / RK3568 / RK3528 / RK3506
- 脚本库（SDK 定制、补丁管理）
- 驱动参考实现
- rootfs 部署方案
- QEMU 模拟支持
- CI 流水线

## 前置知识

- Linux 内核开发经验
- 嵌入式 Linux BSP 基础
- Shell 脚本编程

## 简介

面向 NXP i.MX6ULL 的嵌入式 Linux 开发工坊。提供 Docker 一键搭建的开发环境，采用双内核策略（NXP BSP 6.12.3 + Mainline 7.0rc），覆盖从交叉编译工具链到 U-Boot、内核开发、根文件系统、驱动开发的完整学习路径。

解决了传统嵌入式 Linux 教程"步骤缺失"的痛点——5 分钟 Docker 即可启动完整开发环境，WSL2 原生支持。

## 核心特性

- Docker 一键环境搭建，5 分钟就绪
- 双内核策略：BSP 6.12.3 + Mainline 7.0rc
- 43+ 驱动教程（GPIO、I2C、SPI、Input 等）
- 完整 CI/CD 流水线
- WSL2 支持
- 正点原子阿尔法开发板适配

## 前置知识

- C 语言编程基础
- Linux 基本命令操作
- 对嵌入式硬件有基本了解

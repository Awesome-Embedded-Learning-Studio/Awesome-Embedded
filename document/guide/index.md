| 想学 C++ 的嵌入式开发者 | [Tutorial_AwesomeModernCPP](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_AwesomeModernCPP) — 从 C++11 到 C++23 的系统教程 |
| 想学 Qt / GUI 开发 | [Tutorial_AwesomeQt](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_AwesomeQt) — Qt 6 百科全书式教程 |

## 五条主线

我们的学习和建设围绕五条主线展开。

### 1. Modern C++

从手搓 INI 解析器到现代 C++ 全栈教程，目标是让嵌入式开发者和任何 C++ 初步入门的爱好者也能写出工程级的 C++ 代码。

- [Tutorial_AwesomeModernCPP](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_AwesomeModernCPP) — 现代 C++ 系统教程（C++11~C++23）
- [Project_CXXBaseComponents](https://github.com/Awesome-Embedded-Learning-Studio/Project_CXXBaseComponents) — B 站动画系列 C++ 基础组件源码
- [miniwget](https://github.com/Awesome-Embedded-Learning-Studio/miniwget) — 轻量级 HTTP 下载工具，C++ 工程化教学项目（敬请期待）
- [Tutorial_cpp_SimpleIniParser](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_cpp_SimpleIniParser) — 手搓最简 INI 解析器
- [Project_MakeAMemroyPool](https://github.com/Awesome-Embedded-Learning-Studio/Project_MakeAMemroyPool) — 配合视频的内存池实战
- [CFBox](https://github.com/Awesome-Embedded-Learning-Studio/CFBox) — C++23 极简 BusyBox（也属系统软件线）

### 2. Embedded Linux

从 i.MX6ULL BSP 到 RK 芯片适配，从内核到用户空间的动手实验。

- [imx-forge](https://github.com/Awesome-Embedded-Learning-Studio/imx-forge) — i.MX6ULL BSP 补丁快速部署
- [rk-forge](https://github.com/Awesome-Embedded-Learning-Studio/rk-forge) — Rockchip 系列 BSP 补丁
- [PenguinLab](https://github.com/Awesome-Embedded-Learning-Studio/PenguinLab) — 嵌入式 Linux 全栈动手实验（也属系统软件线）

### 3. Linux / System Programming

面向 Linux 用户态、内核理解、OS 实验与 Unix 工具的系统软件线。

- [Cinux](https://github.com/Awesome-Embedded-Learning-Studio/Cinux) — 现代 C++17 x86_64 操作系统实验项目，Bootloader 到 GUI 全链路
- [CFBox](https://github.com/Awesome-Embedded-Learning-Studio/CFBox) — C++23 Unix 工具集 + userspace 实验载体
- [PenguinLab](https://github.com/Awesome-Embedded-Learning-Studio/PenguinLab) — QEMU 内核与调试实验

### 4. MCU / 裸机 / FreeRTOS

STM32 HAL 驱动、裸机驱动库、FreeRTOS 教程，专注资源受限平台。

- [ST-Forge](https://github.com/Awesome-Embedded-Learning-Studio/ST-Forge) — STM32 HAL 驱动框架，CMake 原生构建
- [BareMetal-Drivers](https://github.com/Awesome-Embedded-Learning-Studio/BareMetal-Drivers) — 单片机裸机通用驱动库
- [Project_MicroWatch](https://github.com/Awesome-Embedded-Learning-Studio/Project_MicroWatch) — 资源受限平台智能手表
- [Tutorial_FreeRTOS](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_FreeRTOS) — FreeRTOS 系列教程（正在规划激活中）
- [micro-forge](https://github.com/Awesome-Embedded-Learning-Studio/micro-forge) — ARM Cortex-M3 (STM32F103) 全系统模拟器，无需硬件即可运行和调试固件

### 5. Qt / GUI / 产品化

从 Qt 教程到嵌入式桌面框架，把学习成果变成可以展示的产品。

- [Tutorial_AwesomeQt](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_AwesomeQt) — Qt 6 百科全书式教程
- [CFDesktop](https://github.com/Awesome-Embedded-Learning-Studio/CFDesktop) — 嵌入式设备 Material Design 3 桌面框架
- [qt-compile-pipeline](https://github.com/Awesome-Embedded-Learning-Studio/qt-compile-pipeline) — Qt6 ARM 交叉编译自动化管道

### 实验项目

这些项目处于早期探索阶段：

- [edgecv](https://github.com/Awesome-Embedded-Learning-Studio/edgecv) — 嵌入式边缘计算机视觉库，Header-Only
- [bareline](https://github.com/Awesome-Embedded-Learning-Studio/bareline) — 轻量级 CLI 交互框架（敬请期待）
- [Tutorial_EmbeddedCommonTools](https://github.com/Awesome-Embedded-Learning-Studio/Tutorial_EmbeddedCommonTools) — 嵌入式开发通用工具教程（正在规划激活中）

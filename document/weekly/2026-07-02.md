## 一句话总览

这两周(本期跨度 11 天——W26 没单独发刊,数据合并到这期一起算)最实的两件事:**Cinux 系把「造操作系统 + GUI + 配套书」一条龙全速往前推,主线就一个——尽快在自己 OS 上把 GCC 跑起来**;**CF 系(coreutils 重写 + 桌面 + 壁纸)也各自上了一个台阶**。forge 家族多了 H618 新成员,教程线多点开花。

## 本周开发进展

先说最猛的 **Cinux 系**。Cinux 本体这两周在疯狂加速——ext4 extents 读路径、TmpFs 内存文件系统、SysV 共享内存、mount/umount、sys_access/poll/select 一路铺,所有动作都围着同一个目标转:**尽快在 OS 上跑起 GCC 编译(GCC 自举)**。配套的 **Cinux-GUI** 同步迭代到 P5:per-widget dirty、flex 权重布局、ANSI 彩色终端、主题变体、字体整数缩放,fbdev-host 切到了控件树;**Cinux-Book** 也跟着重写教程、对齐 VitePress 构建。一条龙三件套齐头并进。

**CF 系**这边,**CFBox** 这周是「性能周」——md5sum 流式 64K 分块、tar 流式归档、sed 正则预编译(~4x)、cmp lockstep 早退、行读取快了 ~7x,还接上 google-benchmark 和 banned-pattern 结构门禁;顺带修了 armhf 32 位下 `-Wconversion` 的坑、去掉 `<charconv>` 防 CI 交叉编译 OOM。**CFDesktop** 把双模式 App 框架立起来了(app.json manifest 发现、launcher 搜索、移植了 SystemState / AlarmyClock / Calendar / Noter、壁纸动画轮播引擎),开始有桌面生态的样子。

**forge 家族**迎来新成员 **h618_forge**——AllWinner H618 LubanCat-A1,pure mainline 的 U-Boot / Linux / rootfs / HDMI 加完整教程,和 rk-forge / imx-forge 凑齐三家。rk-forge 这两周动静不小:RKFW SD-card 启动跑通(纯 ext4 root,板级验证),rootfs 构建顺手重了一刀——工具链路径从 `toolchain.conf` 推导、不再依赖 `PATH`,WSL 下的检测也加强了;还搭起 VitePress 站点(部署工作流镜像 imx-forge、切到 pnpm),收掉 issue #6,也顺势成了本期涨星最多的仓(+12)。imx-forge 补了 RTC 和 GT911 触摸驱动。三家还统一上了 pandoc + WeasyPrint 全站 PDF 导出。

教程线也没闲:**TAMCPP** 发了 v0.7.0,vol3 迁移收尾,补了 networking / 包管理器 / 设计模式几章;**Tutorial_AwesomeQt** 的 widget 栏 13/13 收齐(ip-edit、fade-animation、editable-table、checkbox-tree、speed-meter 自绘那批);**PenguinLab** 一次性铺了 drivers / debugging / networking 三条藤、各 8-9 篇,基建坑也修了一轮。还有 **C-Journey** 出了波大的——整个 C 学习旅程重写,8 阶段路线图统一、整合了维护者自己的代码 lab,新上了 VitePress 站点(部署在 /C-Journey/ 子路径,Hero 终端那套视觉从 TAMCPP 移植过来)。

本站 **Awesome-Embedded** 这两周在悄悄做双语化,给英文版铺路。

## 组织动态

新来了三个公开仓库:**h618_forge**、**Cinux-GUI**(从 Cinux 系拆出来的 GUI 框架)、**CF-Gallery**(CFDesktop 的壁纸资源库,首版 68 张照片,PhotoSwipe 灯箱 + 瀑布流,还能一键装进 CFDesktop 当壁纸包)。

顺带一提,这批仓库这两周统一把 GitHub Pages 部署从 peaceiris/gh-pages 迁到了 actions/deploy-pages——前者基本停更了,换官方部署链路更稳。

## ✨ 我们的目标（短句）

- 做“能学到东西”的小玩具，而不是仅仅展示工程；
- 每个项目要有完整的入门文档、PCB、固件和教学资料；
- 建立一套小而可复用的 Repo Template，方便快速起步；
- 鼓励开源与分享，让更多同好者能参与进来。
